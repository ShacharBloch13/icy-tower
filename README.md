# Shachar's Icy Tower
``` https://youtu.be/fyprmCoYuNw ```

Welcome to Shachar's Icy Tower, a dynamic, browser-based jumping game. In this game, the player must navigate a series of platforms while avoiding enemies and collecting power-ups to achieve the highest score possible.

# Game Features
1) Dynamic Platforms: Platforms are generated with varying properties, including movement and special abilities.
2) Enemies and Power-Ups: Different types of enemies challenge the player, and power-ups like stars can be collected for temporary invincibility.
3) Scoring System: Points are earned by jumping on new platforms and collecting special items.
4) High Score Tracking: The game tracks the top ten scores achieved during the session.

``` git clone https://github.com/ShacharBloch13/icy-tower ```
``` open index.html ```

# How to play
1) Use the arrow keys to move the player left or right.
2) Press space to jump.
3) Avoid enemies and try to land on platforms to continue jumping higher.
4) Collect stars to not die when touching an enemy and extra points to increase your score.
5) The game's background color changes dynamically based on your score to reflect different 'levels' or stages of the game.

# Controls
1) Left Arrow: Move left
2) Right Arrow: Move right
3) Spacebar: Jump

# Technology used
1) HTML5: The game is built using HTML5, with the <canvas> element at its core. This allows for dynamic, scriptable rendering of 2D shapes and bitmap images.
2) JavaScript: All game logic, including player movement, collision detection, platform generation, and score management, is implemented in JavaScript, making it possible to run the game in any modern web browser without additional plugins.
